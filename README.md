# Expense Manager

A clean, modern web application to help you track shared expenses and manage debts with friends and family.

## Features

- **Add People**: Easily add participants to your expense tracking group.
- **Track Expenses**: Log expenses detailing the amount, who paid, and who it was for.
- **Smart Calculation**: Automatically calculates how much each person owes or is owed.
- **Real-time Dashboard**: View current balances and recent activity at a glance.
- **Transaction Details**: Click on any expense to see full details including date and time.
- **Data Persistence**: Your data is saved locally in your browser, so you can come back later.

## Live Preview

Check out the live version here: [Expense Manager Demo](https://teramerahesaab.netlify.app/)

## How to Use

1.  **Open the Dashboard**: Start by visiting the [Live Preview](https://teramerahesaab.netlify.app/) or opening `index.html` locally.
2.  **Add People**: Click "Add Person" to register everyone involved.
3.  **Add Expense**: Click "Add Expense", enter the details, and let the app handle the math.
4.  **Settle Up**: Use the dashboard to see who owes whom.

For a more detailed visual guide, click on the **How to Use** link in the dashboard header.
